# 🎓 CampusKart

CampusKart is a full-stack marketplace designed exclusively for college students to buy and sell used study essentials within their campus community. It provides a simple, secure, and user-friendly platform where students can list products, browse available items, and manage their own listings.

---

## 🚀 Features

- 👤 User Registration & Login (JWT Authentication)
- 🛡️ Protected Routes
- 📦 Add New Products
- 📝 Edit Existing Products
- ❌ Delete Products
- 📋 My Listings Dashboard
- 🔍 Search Products
- 🏷️ Category-wise Product Filtering
- 🖼️ Product Image Upload
- 📱 Fully Responsive UI
- 💬 Beautiful Alerts using SweetAlert2

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/PrachiSharma8/CampusKart-.git
```

### Go to project directory

```bash
cd CampusKart
```

### Install backend dependencies

```bash
cd backend
npm install
```

### Create `.env`

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_CONNECTION_STRING
JWT_SECRET=YOUR_SECRET_KEY
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

Open `index.html` using Live Server in VS Code.
